# Voice (aka gcw_mic)

GCW Microphone Recorder and Player based on [aplay][1] by Jaroslav Kysela.

#### Requirements

- SDL
- SDL_ttf
- SDL_image
- pthread
- ALSA
- C++11
- GCW toolchain to build for GCW

#### Control Keys

- **A** - Start/Stop recording
- **B** - Start/Stop playing
- **UP/DOWN** - Up/Down volume
- **SELECT** - Close
